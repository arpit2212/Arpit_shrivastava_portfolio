# 🚀 Arpit Shrivastava | Full Stack Developer Portfolio

Welcome to my personal portfolio! This project is a high-performance, visually stunning, and interactive web application designed to showcase my skills, projects, and professional journey as a Full Stack Developer and Software Engineer.

![Portfolio Preview](src/assets/hero.png)

## ✨ Key Features

- **🌌 Dynamic Background**: Immersive Starfield background using WebGL (via [ogl](https://github.com/o-g-l/ogl)) for a futuristic feel.
- **🖱️ Interactive UI**: Custom cursor effects and magnetic button animations for enhanced user engagement.
- **🎭 Smooth Animations**: Powered by [Framer Motion](https://www.framer.com/motion/) for fluid transitions and scroll-based animations.
- **⌨️ Typing Effect**: Dynamic tagline rotation with a sleek typing animation in the Hero section.
- **📱 Fully Responsive**: Built with [Tailwind CSS 4](https://tailwindcss.com/) to ensure a seamless experience across all devices.
- **🛠️ Modern Tech Stack**: Utilizing the latest versions of React (v19) and Vite (v8) for optimal performance.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/arpit-shrivastava/Arpit_shrivastava_portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Arpit_shrivastava_portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to see the result.

### Build
Build the project for production:
```bash
npm run build
```
